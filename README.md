## go-httpserver

使用go语言开发的一款快速开启http服务的工具 , 用于在内网不出网中文件的下载传输

使用默认端口 7788

```
httpserver.exe
```

![image-20220602105530361](https://picgo-1301783483.cos.ap-nanjing.myqcloud.com/image/image-20220602105530361.png)

指定端口 -p

```
httpserver.exe -p 8897
```
